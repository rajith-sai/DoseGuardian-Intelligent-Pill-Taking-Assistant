# DoseGuardian: Intelligent Pill-Taking Assistant 💊

## 📌 Project Description
**DoseGuardian** is an ARM-based embedded system designed to assist patients in taking their medications on time. The system provides timely reminders using alerts and display notifications to ensure proper medication adherence.

This project aims to solve a common healthcare issue where people forget to take medicines at the correct time, which may lead to health complications.

---

## 🎯 Objectives
- Develop a smart medicine reminder system.
- Implement an embedded healthcare application using an ARM microcontroller.
- Provide timely alerts and notifications for medication schedules.
- Improve patient discipline in medication intake.

---

## 🚀 Features
- Medicine reminder alerts
- Buzzer notification for scheduled dose time
- LED indicator for visual alert
- LCD display for reminder messages
- Real-time monitoring using RTC
- User acknowledgment button

## ⚙️ Working Principle
1. The user sets the medicine schedule in the system.
2. The RTC module keeps track of the current time.
3. When the scheduled time arrives:
   - The buzzer generates an alert.
   - The LED turns ON.
   - The LCD displays a reminder message.
4. The user acknowledges the reminder by pressing a button.
5. The system waits for the next scheduled medication time.
